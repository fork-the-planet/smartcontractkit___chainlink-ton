# Router User Interface

The Router is the public entrypoint for users to interact with the CCIP system. It provides a unified interface for both sending and receiving messages across chains.

The Router's user interface can be categorized into two flows:

1. [Sender User Interface](./sender.md): This flow covers how users can send messages across chains using the Router.
2. [Receiver User Interface](./receiver.md): This flow covers how users can receive messages from other chains using the Router.
